# Pac-Man (Arcade-Style Clone)

Browser Pac-Man clone using plain HTML, CSS, and JavaScript with original-style mechanics.

## Run

Open `index.html` in your browser.

If you prefer a local server:

- `python -m http.server 8080`
- Then open `http://localhost:8080/pacman/`

## Controls

- Arrow keys
- WASD

## Implemented

- Classic-style maze with pellets and power pellets
- Tunnel wrapping across left and right edges
- Scatter/chase cycle timings
- Frightened mode + ghost combo scoring (200/400/800/1600)
- Four ghosts with personality-inspired targeting behavior
- Ghost house release timing and eye return after being eaten
- 3 lives, level clear reset, game over, restart button
