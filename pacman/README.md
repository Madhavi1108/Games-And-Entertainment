# 🟡 Pacman Game (JavaScript)

A classic **Pacman game** built using **HTML, CSS, and JavaScript**, recreating the nostalgic arcade experience directly in the browser.

---

## 🚀 Features

* 🎮 Smooth keyboard-based controls
* 👻 Ghost enemies with basic movement logic
* 🍒 Food/pellet collection system
* 💯 Score tracking
* 🧱 Grid-based game layout
* 🔄 Restart functionality

---

## 🛠️ Tech Stack

* **HTML5** – Structure of the game
* **CSS3** – Styling and layout
* **JavaScript (Vanilla)** – Game logic and interactivity

---

pacman/

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/Games-And-Entertainment.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Games-And-Entertainment/pacman
   ```

3. Open `index.html` in your browser
   (Double-click or use Live Server in VS Code)

## 🎮 Controls

| Key         | Action     |
| ----------- | ---------- |
| Arrow Up    | Move Up    |
| Arrow Down  | Move Down  |
| Arrow Left  | Move Left  |
| Arrow Right | Move Right |

---

## 🧠 Game Logic Overview

* The game grid is represented as an array
* Pacman moves within boundaries and collects food
* Ghosts move automatically inside the grid
* Collision detection:

  * Pacman + food → score increases
  * Pacman + ghost → game over
